# YAC = Yet Another Cron

This is a super-simpliffied cron, that run tasks every `n` minutes (where `n`) is defined in a configuration JSON file...

## Install

- From sources
    ```bash
    go get -u -v github.com/skryde/yac
    $GOPATH/bin/yac  # run yac.
    ```
